Welcome

This is an API I built for my joetwebdev demo projects.

Currents routes available are:

demo.joetwebdev.io/employees

This first route will return a data table with "employees". It will grab all employees in that table.

demo.joetwebdev.io/employees?page=3&limit=15

A better option for grabing this data would be to include queries.

The query that is used is page and limit. Page will be the page number and limit will be how many employees to include in a page.

So an example would be: demo.joetwebdev.io/employees?page=1&limit=20

This would grab 20 entries for page 1. Page 2 would be the next 20 entries after the 1st page.
